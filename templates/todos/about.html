Using the API

Hereâ€™s how to use the Todos API in different programming languages:

Python Example

import requests

url = 'http://localhost:8000/api/todos/'
headers = {'APIKEY': 'your_secure_api_key'}

# GET todos
response = requests.get(url)
print(response.json())

# POST todo
data = {'title': 'New Todo', 'description': 'Todo description'}
response = requests.post(url, json=data, headers=headers)
print(response.json())

JavaScript Example

fetch('http://localhost:8000/api/todos/', {
    method: 'GET',
    headers: {'APIKEY': 'your_secure_api_key'}
})
.then(response => response.json())
.then(data => console.log(data));

cURL Example

# GET request
curl -H "APIKEY: your_secure_api_key" "http://localhost:8000/api/todos/"

# POST request
curl -X POST -H "Content-Type: application/json" -H "APIKEY: your_secure_api_key" -d '{"title": "New Todo", "description": "Todo description"}' "http://localhost:8000/api/todos/"
